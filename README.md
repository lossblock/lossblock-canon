# LossBlock Canon

This repository contains the canonical ontological foundation of the
**LossBlock Decision Intelligence System**.

The documents in this repository define:

- what truth is (Ontology Canon)
- where truth resides (Layer Model Canon)
- which human psychological failures cause collapse (Tier 0 Canon)
- how canonical truth is governed and protected (Governance Canon)
- where LOSSBLOCK is applicable and where it is not (Scope Canon)

All documents in this repository are **CANONICAL** and **IMMUTABLE**
within their major version.

---

## What This Repository IS

- An ontological constitution for the LOSSBLOCK system
- A formal definition of reality, causality, and decision authority
- A protection layer against human cognitive distortion
- A foundation for all LOSSBLOCK products, engines, and interfaces

---

## What This Repository IS NOT

This repository does **NOT** contain:

- implementation code
- algorithms or thresholds
- decision engines or scoring logic
- optimization methods
- product interfaces or APIs

This repository defines **truth and constraints**, not execution.
